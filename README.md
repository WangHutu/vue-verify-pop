# vue-verify-pop

自带气泡提示的 vue 校验插件

## 示例

```
<!--待校验元素的pop容器-->
<pop>
  <!--该输入框内容为最多为两位小数的数字-->
  <input v-verify decimal-length="2">
</pop>
```

![image](https://github.com/aweiu/vue-verify-pop/raw/master/example.png)

ok，您已经完成了一个基础校验。气泡提示怎么样？丑的话自己用 css 改吧。。
